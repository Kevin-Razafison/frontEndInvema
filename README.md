# frontEndInvema
This is the graphical Interface for the Invema Project
The Invema is an WebApp that meant to manage stock for all categories of things design to be adaptative to all
the users needs

It have many functionnality list up here below:
-Authentification for the admin and for the simple User

-ADMIN ACCOUNT:
 The Admin account have the plenty right to all the users such as:
    *Adding or Deleting an Users
    *Changing All Users passwords
    *Changing All Users Name
    *Accepting a request from other users
    *Adding and deleting stock category
    *Adding and deleting supplier
    *Requesting to the supplier More Items
    *Adding and Deleting products

-SIMPLE USERS:
    *Can Do request to the admin about the products that is low on the stocks
    *Watch in real time All the products
